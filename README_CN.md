# TabWeaver —— 让你的浏览器标签页“活”起来

> **基于 LLM 与 A2UI 的智能标签页整合看板**

---

## 🚀 为什么需要 TabWeaver？

你是否经常在浏览器中打开几十个标签页，却发现自己迷失在信息海洋中？
- **信息过载**：难以快速分辨哪些页面是相关的，哪些是可以关闭的。
- **碎片化阅读**：在不同标签页间反复横跳，无法形成系统的知识逻辑。
- **难以整理**：手动归档极度耗时，最终只能无奈地把它们全部关掉。

**TabWeaver** 为此而生。它不是一个简单的标签管理器，而是一个**智能知识整合助手**。

## ✨ 核心功能

### 1. 🤖 智能聚类 (Topic Clustering)
不再手动分类。利用 LLM 的语义分析能力，TabWeaver 会自动扫描你当前打开的所有页面，并根据其**核心主题和细分领域**进行精准聚类。你得到的不再是一堆链接，而是结构化的主题看板。

### 2. 📝 深度合成 (Deep Synthesis)
一键开启深度分析模式。系统会跨页面提取关键信息，输出详尽的“逐页详解”与“综合分析”。它能帮你发现不同页面之间的**共识、差异与洞见**，将碎片信息转化为系统知识。

### 3. 🗺️ 内容可视化 (Flowcharting)
集成 Mermaid 渲染引擎，为每个页面自动生成**内容流向图**。通过可视化图表快速直观地理解网页结构与复杂逻辑。

### 4. 🧹 极简管理 (Clean & Focus)
在看板中完成阅读和整理后，支持一键关闭该主题下的所有标签页。保持浏览器清爽，让注意力回归核心任务。

## 🛠️ 安装与配置

### 1. 加载插件
1. 下载本项目代码。
2. 打开 Chrome 浏览器并进入 `chrome://extensions/`。
3. 开启右上角的“开发者模式”。
4. 点击“加载已解压的扩展程序”，选择本项目根目录。

### 2. 配置 API
1. 点击插件图标打开 TabWeaver 看板。
2. 在底部的设置区域输入你的 API 配置：
   - **Endpoint**: API 地址（支持 Gemini 或 OpenAI 兼容接口）。
   - **Model**: 使用的模型名称（如 `gemini-1.5-flash`）。
   - **API Key**: 你的私有密钥。
3. 点击“刷新”即可开始智能扫描。

## 🔧 技术栈

- **Manifest V3**: 最新标准的 Chrome 扩展架构。
- **Readability.js**: 网页核心内容提取引擎。
- **Mermaid.js**: 强大的图表渲染引擎。
- **LLM 驱动**: 支持多种大模型（Gemini/DeepSeek/OpenAI 等）。
- **原生 JS/CSS**: 零重力设计，极致轻量与流畅。
